# Junit

## 什么是Junit

JUnit is a simple framework to write repeatable tests. It is an instance of the xUnit architecture for unit testing frameworks.
>>Junit是一个能够简单的编写重复测试的框架，他是XUnit的一个实例。from:[Junit](https://junit.org/junit4/)
